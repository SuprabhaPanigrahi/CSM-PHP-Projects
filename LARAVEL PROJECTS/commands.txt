Fresh + seed

Run specific migration path
    php artisan migrate --path=/database/migrations/myfile.php

Create seeder
    php artisan make:seeder UserSeeder

Run seeder
    php artisan db:seed
    php artisan db:seed --class=UserSeeder

Create controller
    php artisan make:controller UserController

Create middleware
